# Assets — Documentation du projet

## Image des maquettes

Placer dans ce dossier l’image des maquettes (grille des écrans Student Deals / CampusPass), par exemple :

- **maquettes-app-students-deals.png**

Cette image est référencée dans `PROJET_PROPOSITION.md`.

Si l’image est déjà enregistrée ailleurs (ex. dans les assets Cursor), vous pouvez la copier ici pour garder toute la documentation du projet au même endroit.
