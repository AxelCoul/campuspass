IMAGE DES MAQUETTES
-------------------

Placer dans ce dossier l'image des maquettes (grille des écrans Pass Campus),
par exemple :

  maquettes-pass-campus.png

Cette image est référencée dans PROJET_PROPOSITION.txt.

Si l'image est déjà enregistrée ailleurs (ex. dans les assets Cursor), vous
pouvez la copier ici pour garder toute la documentation du projet au même
endroit.
