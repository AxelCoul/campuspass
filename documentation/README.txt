================================================================================
  DOCUMENTATION DU PROJET — Plateforme de réductions étudiantes
================================================================================

Ce dossier rassemble la proposition de projet et les pistes d'amélioration.


FICHIERS
--------

  PROJET_PROPOSITION.txt  — Proposition complète : objectifs, acteurs,
                            architecture, fonctionnalités, écrans, parcours,
                            sécurité, technologies, roadmap.

  AMELIORATIONS.txt       — Clarifications et améliorations proposées
                            (fonctionnel, technique, roadmap).

  assets/                 — Dossier pour l'image des maquettes
                            (voir assets/README.txt).


IMAGE DES MAQUETTES
-------------------

Pour inclure l'image des maquettes dans la documentation :

  1. Copiez votre capture d'écran (grille des écrans) dans
     documentation/assets/

  2. Nommez-la par exemple : maquettes-pass-campus.png

Elle sera alors référencée dans PROJET_PROPOSITION.txt.
