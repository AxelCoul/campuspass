# Documentation du projet — Plateforme de réductions étudiantes

Ce dossier rassemble la **proposition de projet** et les **pistes d’amélioration**.

## Fichiers

| Fichier | Description |
|---------|-------------|
| **PROJET_PROPOSITION.md** | Proposition complète : objectifs, acteurs, architecture, fonctionnalités, écrans, parcours, sécurité, technologies, roadmap. |
| **AMELIORATIONS.md** | Clarifications et améliorations proposées (fonctionnel, technique, roadmap). |
| **assets/** | Dossier pour l’image des maquettes (voir `assets/README.md`). |

## Image des maquettes

Pour inclure l’image des maquettes dans la documentation :

1. Copiez votre capture d’écran (grille des écrans) dans `documentation/assets/`.
2. Nommez-la par exemple : **maquettes-app-students-deals.png**.

Elle sera alors référencée dans `PROJET_PROPOSITION.md`.
